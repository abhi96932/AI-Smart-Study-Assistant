# AI Smart Study Assistant (Offline NLP-Based)

## Overview
AI Smart Study Assistant is a Python-based offline AI chatbot that answers student queries using basic Natural Language Processing (NLP) techniques.

This project uses TF-IDF Vectorization and Cosine Similarity to provide intelligent responses from a custom dataset.

## Features
- Offline AI-based question answering
- NLP text preprocessing
- TF-IDF Vectorization
- Cosine Similarity matching
- GUI using Tkinter
- Custom dataset support

## Technologies Used
- Python
- Scikit-learn
- Tkinter
- JSON

## Project Structure
AI-Study-Assistant/
│

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py
